# Business Intelligence Models

Machine learning models applied to real business scenarios. Built with Python, Pandas, Scikit-learn, and Jupyter Notebooks.

## Projects

### 🍽️ Restaurant Demand Forecasting
Predictive model to forecast dish demand in restaurants based on historical sales data. Helps reduce food waste and optimize purchasing.

### 🏢 Multi-Product Business Analysis
Revenue and cost analysis for businesses with multiple product lines. Includes scenario modeling and trend visualization.

### 📈 Single-Product Sales Model
Simplified ML pipeline for small businesses with a single product/service — clean EDA, feature engineering, and regression.

### 🌤️ Climate Impact on Business
Analysis of how weather/climate variables correlate with business performance metrics.

### 💳 Fintech Financial Projections
5-year financial projection model for a Colombian micro-lending fintech startup (TePrestamos). Includes conservative/moderate/optimistic scenarios.

## Tech Stack

- Python 3.x
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook
